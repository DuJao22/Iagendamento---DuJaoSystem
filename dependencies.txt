# Dependências necessárias para o funcionamento

## Dependências Principais Instaladas

### Framework Web
- Flask (framework web principal)
- Werkzeug (utilitários WSGI e proxy)
- gunicorn (servidor web de produção)

### IA e Processamento
- google-generativeai (Google Gemini para chatbot)

### Base de Dados
- SQLite3 (incluído no Python - nativo)
- Nenhuma dependência externa necessária

### Utilitários
- email-validator (validação de emails)
- logging (logs do sistema - nativo do Python)

## Comandos para Instalação Manual

Se precisar instalar em outro ambiente:

```bash
pip install Flask==3.0.0
pip install Werkzeug==3.0.1  
pip install google-generativeai==0.8.3
pip install gunicorn==21.2.0
pip install email-validator==2.1.0.post1
```

## Para SQLite Cloud (migração futura)

Quando migrar para SQLite Cloud, adicionar:

```bash
pip install libsql-client
pip install turso
```

## Variáveis de Ambiente Necessárias
